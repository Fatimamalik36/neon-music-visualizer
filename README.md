# 🎧 Neon Music Visualizer

A modern interactive **3D music visualizer web application** built using HTML, CSS, JavaScript, and Three.js. It reacts in real-time to music and generates dynamic neon-style animations, particles, and audio-driven effects.

---

##  Live Features

*  Play & upload music files
*  Real-time audio frequency analysis
*  3D neon visualizer (sphere, rings, wave effects)
*  Particle system background animation
*  Audio spectrum bars reacting to sound
*  Multiple theme switching (colors)
*  Mouse-controlled camera movement
*  Screenshot capture feature
*  Screen recording support (MediaRecorder API concept)
*  Fully responsive design

---

##  Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Three.js (WebGL)
* Web Audio API
* MediaRecorder API

---

## 🎮 How It Works

1. Upload or select a music file
2. Audio is analyzed using Web Audio API
3. Frequency data drives animations
4. Three.js renders 3D neon visuals in real-time
5. Particles and rings react to bass, mid, and treble

---

## 📸 Screenshot Feature

* Captures current canvas frame
* Uses `canvas.toDataURL()`
* Downloads image as PNG

---

## 🎥 Screen Recording Feature

* Uses `MediaRecorder API`
* Records live visualizer animation
* Saves output as video file
* Enhances user sharing experience
