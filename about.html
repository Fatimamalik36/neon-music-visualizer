About the project

A visualizer built to feel premium

Neon 3D Music Visualizer is a portfolio-grade demonstration of real-time audio analysis paired with WebGL rendering — no frameworks, just the browser platform.

Project overview

The app pairs the Web Audio API's AnalyserNode with a Three.js scene so that bass, mids, and highs each drive a distinct visual behavior — scale, rotation, and particle speed — while GSAP handles the surrounding interface motion.

Technologies

HTML5 · CSS3 · Vanilla JavaScript (ES6) · Three.js · GSAP · Web Audio API. No build step, no frameworks — everything runs directly in the browser.

Feature set

What's inside

Full Player

Play, pause, seek, shuffle, repeat, keyboard shortcuts, and a live equalizer.

3D Scene Modes

Sphere core, frequency bars, neon tunnel, particle galaxy, and a random mode.

Microphone Mode

Feed live mic input straight into the analyser for real-time reactive visuals.

Clip Recording

Capture a short clip of the visualizer canvas and download it instantly.

FAQ

Frequently asked questions

Does it work without uploading a song?

Yes — a generated demo tone plays automatically so the visualizer has something to react to right away.

Is my audio uploaded anywhere?

No. Files are decoded locally in your browser using the Web Audio API and never leave your device.

Can I use my microphone instead?

Yes — Mic Mode on the Visualizer page asks for permission and routes live input into the same analyser.

What browsers are supported?

Any modern browser with WebGL and Web Audio support — recent Chrome, Firefox, Edge, and Safari.
